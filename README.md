Repositório que servirá como 1º Checkpoint da disciplina de WebDev da FIAP

## Resumo do Projeto
A Social Connect é uma plataforma inovadora de conexão entre pessoas e empresas, focada em criar experiências únicas nas redes sociais. Nossa plataforma oferece aos usuários uma experiência única de conexão e comunicação.
